Incendio Cuesta del Ternero 2021 — Análisis de Índices de Severidad con Google Earth Engine

Scripts de Google Earth Engine (GEE) para el análisis de severidad del incendio ocurrido en la Cuesta del Ternero (Comarca Andina, Patagonia Argentina), 
utilizando imágenes Sentinel-2.


Descripción general

Este repositorio contiene los scripts desarrollados para:

Selección de imágenes pre y post incendio
Calcular índices espectrales de severidad del incendio (dNBR, RBR, RdNBR, NBR post incendio, dNDVI, NDVI post incendio, dCSI, CSI post incendio) a partir de imágenes Sentinel-2 (nivel 2A)
Aplicar máscaras de agua y vegetación para reducir el ruido 
Clasificar la severidad del incendio según umbrales de referencia (USGS) para elaborar mapa preliminar con el índice dNBR


Todo el procesamiento se realizó en Google Earth Engine utilizando computación en la nube.
